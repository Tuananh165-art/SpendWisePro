# SpendWise Pro - Week 3 Build

Ứng dụng demo tuần 3 cho đề tài "Ứng dụng quản lý chi tiêu cá nhân".

Chức năng đã hiện thực:
- Quản lý danh mục thu/chi
- Thêm/sửa/xóa giao dịch
- Thiết lập hạn mức chi tiêu theo tháng
- Cảnh báo hạn mức 70%/90%/100%
- Báo cáo theo tháng và theo năm
- Lưu dữ liệu bằng LocalStorage

Cách chạy:
1. Mở file index.html bằng trình duyệt hiện đại.
2. Hoặc dùng Live Server nếu muốn tự động reload.

Kiểm thử logic:
- Chạy `npm test`
